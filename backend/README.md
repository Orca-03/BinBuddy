# Setup Instructions
### 1. Install Python 3.11
On Windows (cmd) with Python Install Manager
```cmd
pymanager install 3.11
```
### 2. Create virtual environment and set up requirements
On Windows (cmd) from root directory
```cmd
cd .\backend
py -3.11 -m venv .venv
.venv\Scripts\activate
pip install -r "requirements.-dev.txt"
```
### 3. Start FastAPI development server
After activating the virtual environemnt
```
fastapi dev
```
# Testing Endpoints
Use a tool such as [Bruno](https://www.usebruno.com/) to test the API. When hosted locally, the API can be reached at `127.0.0.1:8000`
## Requirements
* `binbuddy_model_best.pth` in `.\backend` 
* `binbuddy_classes.json` in `.\backend`
> Generated by running training scripts
## `/api/classify`
Requires the response body to be sent as multipart form data with one file parameter called `file`. Returns the highest confidence class as `category` and a dictionary mapping classes to their confidence score as `conf_scores`.
